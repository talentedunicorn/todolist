# ToDone

> Todo list app with localStorage

# Features

- [x] Add new todo
- [x] Set todo completed status
- [x] Delete todo
- [x] Clear completed todos
- [x] Sync todo to localStorage
- [x] Edit todo text
- [x] Markdown support
